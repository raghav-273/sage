# SAGE — Standards and Guidelines Engine

SAGE is an engineering knowledge platform for exploring technical standards through evidence-backed AI search, hybrid retrieval, and interactive document navigation.

Built for engineering and government workflows, SAGE combines semantic search, full-text retrieval, verified citations, asynchronous document processing, and research workspaces to help users find trustworthy answers across complex technical documentation.

### Highlights

- Django 5 + Django REST Framework
- PostgreSQL + pgvector
- Celery + Redis asynchronous ingestion
- PDF, figure, and diagram extraction
- Vision-based figure captioning
- Local embedding generation
- Hybrid retrieval (Vector Search + PostgreSQL FTS + Reciprocal Rank Fusion)
- Gemini-powered answer generation
- Citation-enforced responses
- Research Sessions
- Authentication and government-style frontend
- Comprehensive test suite (152/152 passing)

SAGE is designed to be more than an AI chatbot—it is an engineering knowledge platform focused on trust, evidence, navigation, and professional research workflows.
